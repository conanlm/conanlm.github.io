hexo使用教程

发表于 2018-05-19 更新于 2019-06-03 分类于 hexo

安装

文档：https://hexo.io/zh-cn/docs/
先安装

  • Node.js

  • Git

    命令行下

    npm install -g hexo-cli

初始化

hexo init <folder>
cd <folder>
npm install

hexo g //generetor的缩写
hexo s //server的缩写

安装next主题

下载： https://github.com/theme-next/hexo-theme-next

将文件解压到themes,文件夹命名next
在_config.yml文件themes修改为next

添加导航栏

home: / 首页
categories: /categories 分类
archives: /archives 归档
tags: /tags 标签
about: /about 关于
commonweal: /404.html 公益404

创建分类

hexo new page "categories"

创建标签

hexo new page "tags"

创建本地搜索引擎

npm install --save hexo-generator-search
  • 修改 站点配置 文件
  • 修改 主题配置文件

将本地博客上传到github

github新建项目 new reposiory

必须为用户名.github.io

配置 SSH

  • 设置用户名

    git config --global user.name "yourname"

  • 设置用户邮箱

    git config --global user.email "yourmail"

  • 本地生成公钥私钥

    ssh-keygen -t rsa -C "你的邮件地址"

    文件夹C:\Users\lm.ssh
    文本打开id_rsa.pub文件

  • 登录 Github，右上角 头像 -> Settings —> SSH and GPG keys —> New SSH key。把公钥粘贴到key中，填好title并点击 Add key。

  • 在gitbash中验证是否添加成功：

    ssh -T git@github.com

    修改配置文件

  • 修改_config.yml

    deploy:
type: git
repo: https://github.com/YourgithubName/YourgithubName.github.io.git
branch: master

建立文档

hexo new [layout] <title>

您可以在命令中指定文章的布局（layout），默认为 post，可以通过修改 _config.yml 中的 default_layout 参数来指定默认布局。

上传到github

  • 先安装一波：

    npm install hexo-deployer-git --save

  • 执行命令:

    hexo clean
hexo generate
hexo deploy

  • 在浏览器中输入http://yourgithubname.github.io
