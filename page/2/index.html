使用批处理文件同时打多个CMD窗口且执行多条命令

发表于 2018-05-19 分类于 cmd
  • 使用批处理文件时，可以用下面方法打打多个CMD窗口，并且每个窗口中执行多条命令。

start 用来启动一个应用
cmd /k 表示cmd后面的命令执行完后不关闭窗口。如果要在执行完成后关闭窗口可以用/c 。详细请使用cmd/?查看
“命令1&&命令2&&..” 将要执行的多条命令使用引号全部包起来，并且在命令间用&&分隔。如果只有一条命令则不用引号也可以。
