2003 dhcp迁移 2012r2

发表于 2018-05-28 分类于 Hyper-V

在2003服务器上导出dhcp配置文件

netsh dhcp server dump > c:\win2003dump.txt

运行下列命令，ip改为服务器ip

netsh Dhcp Server 10.155.155.10 delete Class "默认路由和远程访问类别" "远程访问客户端的用户类别" 525241532e4d6963726f736f6674 0 b
netsh Dhcp Server 10.155.155.10 delete Class "默认 BOOTP 的类别" "BOOTP 客户端的用户类别" 424f4f54502e4d6963726f736f6674 0 b
netsh Dhcp Server 10.155.155.10 delete Class "Microsoft Windows 2000 选项" "Windows 2000 客户端的 Microsoft 供应商特定选项" 4d53465420352e30 1 b
netsh Dhcp Server 10.155.155.10 delete Class "Microsoft Windows 98 选项" "Windows 98 客户端的 Microsoft 供应商特定选项" 4d534654203938 1 b
netsh Dhcp Server 10.155.155.10 delete Class "Microsoft 选项" "适用于 Windows 98 和 Windows 2000 客户端的 Microsoft 供应商特定选项" 4d534654 1 b



netsh Dhcp Server 10.155.155.10 Add Class "默认路由和远程访问类" "远程访问客户端的用户类" 525241532e4d6963726f736f6674 0 b
netsh Dhcp Server 10.155.155.10 Add Class "默认的网络访问保护级别" "受限访问客户端的默认特殊用户类" 4d5346542051756172616e74696e65 0 b
netsh Dhcp Server 10.155.155.10 Add Class "默认 BOOTP 类" "BOOTP 客户端的用户类" 424f4f54502e4d6963726f736f6674 0 b
netsh Dhcp Server 10.155.155.10 Add Class "Microsoft Windows 2000 选项" "Windows 2000 客户端的 Microsoft 供应商特定选项" 4d53465420352e30 1 b
netsh Dhcp Server 10.155.155.10 Add Class "Microsoft Windows 98 选项" "Windows 98 客户端的 Microsoft 供应商特定选项" 4d534654203938 1 b
netsh Dhcp Server 10.155.155.10 Add Class "Microsoft 选项" "适用于 Windows 98 和 Windows 2000 客户端的 Microsoft 供应商特定选项" 4d534654 1 b

2003上导出dhcp

netsh dhcp server export c:\dhcp.txt all

在2012r2上导入

netsh dhcp server import c:\dhcp.txt
# dhcp # Hyper-V
